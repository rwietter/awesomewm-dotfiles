## Requirements

- Nerd fonts
- [Termicons fonts](https://mskelton.github.io/termicons/)

## Escape Sequences

- Leader = Space
- A = Alt
- C = Ctrl
- S = Shift
- R = Replace

## Keymap Modes

- Insert = i
- Normal = Esc
- Visual = v
- V-block = Ctrl + v
- V-line = Shift + v
