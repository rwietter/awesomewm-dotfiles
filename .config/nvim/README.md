## Escape sequences

leader = Space
A = Alt
C = Ctrl
S = Shift

## Keymap modes

Insert = i
Normal = Esc
Visual = v
V-block = Ctrl + v
V-line = Shift + v

## Commands

| ------- | ------------------------------------ |

| :q!     | Fecha o arquivo sem salvar           |
| :e      | Abre um arquivo                      |
| :sp     | Abre um novo arquivo                 |
| :vsp    | Abre um novo arquivo na vertical     |
